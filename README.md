# 女性成长公众号运营资料库

这是一个围绕 **30岁+女性清醒成长** 搭建的公众号运营资料库。

## 核心定位

**30岁+女性清醒成长**

目标人群：

- 30—45岁女性为主
- 25—50岁女性为覆盖范围

核心主线：

> 少耗自己，有钱有退路。

## 内容方向

- 婚姻清醒：关系、夫妻、家庭分工
- 赚钱独立：收入、存钱、现金流
- 情绪止损：不内耗、不讨好
- 关系边界：亲密关系与人际消耗
- 中年觉醒：30+女性人生重建
- 人生退路：安全感、底线、后路
- 低消耗生活：减少消耗、稳定生活
- 职场止损：烂工作、转行焦虑、离职底气
- 自我重建：精气神、打理自己、后半场状态

## 账号风格

- 清醒但不刻薄
- 现实但不焦虑
- 温柔但不软弱
- 有刺痛感，但必须给出口
- 不煽动对立，不极端情绪化
- 不写空洞鸡汤
- 不做说教型文章
- 少用三联排比和机械口号，多用生活动作和自然表达

## 五大爆文模型

1. 气质人格型：气正、靠谱、干净、有度、有主心骨
2. 催婚婚姻清醒型：催婚、将就、嫁错、父母焦虑
3. 存款安全感型：存钱、退休、低成本生活、后半场退路
4. 烂工作及时止损型：工作内耗、失眠、身体报警、离职底气
5. 中年自我重建型：打扮、精气神、悦己、重新认真对待自己

## 最高优先级表达规则

写作时必须优先执行：

> 少用三联排比，少用机械口号，多用生活动作和自然表达。

如果一段话看起来很整齐，但不像真人说话，就删掉重写。

## 账号记忆点

读者必须反复记住这句话：

> 少耗自己，有钱有退路。

这个公众号不是情绪散文号，也不是鸡汤号，而是写给30岁以后普通女人的清醒成长号。

最终目标：

> 写普通女人怎么在30岁以后，少受委屈，少内耗，有钱花，有路退，有日子过。
