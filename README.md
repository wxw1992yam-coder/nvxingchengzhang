# 女性成长公众号运营资料库

这是一个围绕 **30岁+女性清醒成长** 搭建的公众号运营资料库。

## 核心定位

**30岁+女性清醒成长**

目标人群：

- 30—45岁女性为主
- 25—50岁女性为覆盖范围

核心主线：

> 少耗自己，有钱有退路。

## 内容方向

- 婚姻清醒：关系、夫妻、家庭分工
- 赚钱独立：收入、存钱、现金流
- 情绪止损：不内耗、不讨好
- 关系边界：亲密关系与人际消耗
- 中年觉醒：30+女性人生重建
- 人生退路：安全感、底线、后路
- 低消耗生活：减少消耗、稳定生活

## 账号风格

- 清醒但不刻薄
- 现实但不焦虑
- 温柔但不软弱
- 有刺痛感，但必须给出口
- 不煽动对立，不极端情绪化
- 不写空洞鸡汤
- 不做说教型文章

## 仓库文件说明

- `01-project-instructions.md`：可复制到 ChatGPT 项目指令里的完整运营指令
- `02-baowen-analysis-skill.md`：爆文拆解 Skill 标准
- `03-writing-system.md`：账号写作系统与文章结构
- `04-seven-day-launch-plan.md`：7天起号内容计划
- `05-cover-visual-system.md`：公众号视觉模板与封面标题系统
- `06-title-bank.md`：可直接使用的标题库
- `07-day1-article.md`：第一天定调文优化版

## 账号记忆点

读者必须反复记住这句话：

> 少耗自己，有钱有退路。

这个公众号不是情绪散文号，也不是鸡汤号，而是写给30岁以后普通女人的清醒成长号。
